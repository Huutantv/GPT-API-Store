Hướng Dẫn Cài Đặt

Tương thích OpenAI SDK, Cline, Roo Code, OpenClaw, Codex CLI

Bắt đầu nhanh - Python OpenAI SDK

1. Cài đặt

pip install openai

2. Code Python

from openai import OpenAI

client = OpenAI(
    api_key="sk-your-api-key",
    base_url="https://your-domain.com/v1"
)

response = client.chat.completions.create(
    model="gpt-5.5",
    messages=[{"role": "user", "content": "Xin chào!"}],
    stream=True
)

for chunk in response:
    print(chunk.choices[0].delta.content or "", end="")

3. Test bằng cURL

curl https://your-domain.com/v1/chat/completions \
  -H "Authorization: Bearer sk-your-api-key" \
  -H "Content-Type: application/json" \
  -d '{"model":"gpt-5.5","messages":[{"role":"user","content":"Hi"}]}'

Cài vào Cline / Roo Code (VS Code)

1

Cài extension Cline hoặc Roo Code trong VS Code.

2

Mở Settings của extension.

3

Chọn API Provider: OpenAI Compatible. Không chọn Anthropic.

4

Nhập Base URL: https://your-domain.com/v1

5

Dán API Key vào ô API Key.

6

Nhập model: gpt-5.5 hoặc gpt-5.5-turbo

7

Gửi thử một prompt ngắn để kiểm tra.

Provider : OpenAI Compatible
Base URL : https://your-domain.com/v1
API Key  : sk-your-api-key
Model    : gpt-5.5
⚠️ Không tự bịa model. Phải chọn đúng từ danh sách: gpt-5.5, gpt-5.5-turbo, gpt-5.4, gpt-4o

Cài vào OpenClaw

1

Trong OpenClaw, mở cấu hình model provider và thêm provider mới, đặt id: gptapi

2

Chọn API type: openai-completions

3

Nhập Base URL và API Key.

4

Khai báo models và dùng dạng gptapi/gpt-5.5 khi chọn.

{
  models: {
    providers: {
      gptapi: {
        api: "openai-completions",
        baseUrl: "https://your-domain.com/v1",
        apiKey: "sk-your-api-key",
        models: [
          { id: "gpt-5.5" },
          { id: "gpt-5.5-turbo" },
          { id: "gpt-5.4" }
        ]
      }
    }
  },
  agents: {
    defaults: {
      model: { primary: "gptapi/gpt-5.5" }
    }
  }
}

Cài vào Codex CLI

1

Export biến môi trường trong terminal:

export OPENAI_API_KEY="sk-your-api-key"
export OPENAI_BASE_URL="https://your-domain.com/v1"
2

Chạy Codex với --model:

codex --model gpt-5.5 "Viết giúp tôi một hàm JavaScript debounce"
⚠️ Nếu lệnh env không ăn, kiểm tra config file của bản Codex (~/.codex/config.toml) nhưng giữ nguyên 3 giá trị: API key, Base URL, model.
