Hướng Dẫn Cài Đặt

Tương thích OpenAI SDK, Cline, Kilo Code, OpenClaw, Codex CLI

Bắt đầu nhanh - Python OpenAI SDK

1. Cài đặt

pip install openai

2. Code Python

from openai import OpenAI

client = OpenAI(
    api_key="sk-your-api-key",
    base_url="https://zplay.io.vn/v1"
)

response = client.chat.completions.create(
    model="gpt-5.5",
    messages=[{"role": "user", "content": "Xin chào!"}],
    stream=True
)

for chunk in response:
    print(chunk.choices[0].delta.content or "", end="")

3. Test bằng cURL

curl https://zplay.io.vn/v1/chat/completions \
  -H "Authorization: Bearer sk-your-api-key" \
  -H "Content-Type: application/json" \
  -d '{"model":"gpt-5.5","messages":[{"role":"user","content":"Hi"}]}'

Cài đặt Cline / Kilo Code (VS Code)

1

Cài extension Cline hoặc Kilo Code

VS Code → Extensions (Ctrl+Shift+X) → Tìm "Cline" hoặc "Kilo Code" → Install

2

Mở Settings → Providers

Cline: mở biểu tượng Cline bên trái → Settings hoặc API Configuration.
Kilo Code: mở biểu tượng Kilo Code bên trái → Settings → Provider / API Provider.

3

Nhập thông tin cấu hình

API Provider OpenAI Compatible
Base URL https://zplay.io.vn/v1
API Key sk-your-api-key
Model gpt-5.5
4

Bật Streaming nếu có

Bật Enable Streaming trong Settings để phản hồi nhanh hơn.

5

Lưu cấu hình và test thử

Click Save → Mở chat Cline hoặc Kilo Code → Gõ thử: Hello, bạn có hoạt động không?

Cấu hình mẫu: API Provider OpenAI Compatible, Base URL https://zplay.io.vn/v1, Model gpt-5.5, Enable Streaming On.
ℹ️ Model có thể chọn: gpt-5.5 · gpt-5.5-turbo · gpt-5.4 · gpt-4o
⚠️ Phải chọn OpenAI Compatible; không chọn Anthropic, OpenAI mặc định, Google Gemini hoặc OpenRouter nếu không có hướng dẫn riêng.

Cài vào OpenClaw

1

Trong OpenClaw, mở cấu hình model provider và thêm provider mới, đặt id: gptapi

2

Chọn API type: openai-completions

3

Nhập Base URL và API Key.

4

Khai báo models và dùng dạng gptapi/gpt-5.5 khi chọn.

{
  models: {
    providers: {
      gptapi: {
        api: "openai-completions",
        baseUrl: "https://zplay.io.vn/v1",
        apiKey: "sk-your-api-key",
        models: [
          { id: "gpt-5.5" },
          { id: "gpt-5.5-turbo" },
          { id: "gpt-5.4" }
        ]
      }
    }
  },
  agents: {
    defaults: {
      model: { primary: "gptapi/gpt-5.5" }
    }
  }
}

Cấu hình Codex CLI

1

Tạo thư mục .codex trong thư mục gốc hệ thống.

macOS/Linux: ~/.codex/
Windows: %USERPROFILE%\.codex\
2

Tạo tệp config.toml:

model_provider = "zplay"
model = "gpt-5" #model = "gpt-5-codex"
model_reasoning_effort = "high"
disable_response_storage = true

[model_providers.zplay]
name = "zplay"
base_url = "https://zplay.io.vn/v1"
wire_api = "responses"
requires_openai_auth = true
3

Tạo tệp auth.json:

{
  "auth_mode": "apikey",
  "OPENAI_API_KEY": "your-api-key-here"
}
Lưu ý: Bạn cần tạo thư mục .codex thủ công, sau đó tạo config.toml và auth.json bên trong.

Có. Bạn chỉ cần thay base_url và api_key. Tương thích hoàn toàn với openai Python/Node SDK.

Hỗ trợ streaming không?+

Có. Hỗ trợ đầy đủ SSE streaming cho OpenAI chat completions format.

Có hỗ trợ function calling / tools không?+

Có. Hỗ trợ đầy đủ function calling tương thích OpenAI format.

Thanh toán bằng gì?+

Chuyển khoản ngân hàng, Momo, ZaloPay. Nhận API key trong 5 phút sau thanh toán.

Dùng được với Cline, Kilo Code, Cursor không?+

Có. Chọn provider OpenAI Compatible, nhập Base URL https://zplay.io.vn/v1, API key và model gpt-5.5 là xong.
